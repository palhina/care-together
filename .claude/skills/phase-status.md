# /phase-status スキル

現在の開発フェーズの進捗を確認・更新します。

## 実行内容

1. CLAUDE.mdの開発フェーズセクションを読み込み
2. 各フェーズのTODO完了状況を確認
3. 現在取り組むべきタスクを提案

## 開発フェーズ一覧

- **Phase 1**: 基盤構築（Expo初期化、Supabase、認証）
- **Phase 2**: オンボーディング（ウェルカム、グループ作成、ペット登録）
- **Phase 3**: コア機能（ホーム画面、日付ナビ、タスク完了、リアルタイム同期）
- **Phase 4**: テンプレート・ペット管理
- **Phase 5**: 統計・グループ機能
- **Phase 6**: 仕上げ（UI/UX、エラーハンドリング、App Store申請）

## 使用例

ユーザー: `/phase-status`

→ 現在のフェーズ進捗と次に取り組むべきタスクを表示
